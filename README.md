## 🛠️ Tech Stack

- **State Management:** Context API
- **Icons:** React Icons

## 📂 Folder Structure
```
📦 inventory-management
├── 📁 src
│   ├── 📁 components      # Reusable UI components (Table, ProductForm, Alert)

│   ├── 📄 App.js          # Main component
│   ├── 📄 index.js        # Entry point
```
